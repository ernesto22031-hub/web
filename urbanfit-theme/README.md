# UrbanFit WordPress Theme

A modern, fully customizable gym and fitness WordPress theme with modular HTML blocks.

## Features

✅ **Fully Customizable Sections**
- Hero section with custom image, title, and subtitle
- All text, buttons, and links customizable from WordPress Customizer
- Colors (primary, text, background) fully customizable
- Logo and site description support

✅ **Modern Design**
- Dark theme with orange/red accents
- Responsive design (mobile, tablet, desktop)
- Smooth animations and transitions
- Professional typography

✅ **WordPress Integration**
- Gutenberg support
- Widget areas in footer
- Custom logo support
- Navigation menus
- Blog post support
- Featured images support

✅ **Developer Friendly**
- Modular structure
- Easy to customize and extend
- Clean HTML/CSS code
- Well-documented functions

## Installation

1. Upload the `urbanfit-theme` folder to `/wp-content/themes/`
2. Activate the theme in WordPress admin
3. Go to Appearance > Customize to customize colors, hero section, and more

## Customization

### Hero Section
Go to **Appearance > Customize > Hero Section** to customize:
- Hero title
- Hero subtitle
- Background image
- Primary button text and URL

### Colors
Go to **Appearance > Customize > Colors** to customize:
- Primary color (orange by default)
- Text color
- Background colors (automatic)

### Navigation
Go to **Appearance > Menus** to create and assign your navigation menu

### Logo
Go to **Appearance > Customize > Site Identity** to upload your custom logo

## Theme Template Files

- `header.php` - Header and navigation
- `footer.php` - Footer with widget areas
- `index.php` - Blog/home page template
- `page.php` - Single page template
- `single.php` - Single post template
- `style.css` - Main stylesheet
- `functions.php` - Theme setup and customizer options

## Creating Custom Templates

You can create additional templates by copying and modifying existing files. Common templates:

- `home.php` - Home page (if different from blog)
- `archive.php` - Archive pages
- `search.php` - Search results
- `404.php` - 404 page
